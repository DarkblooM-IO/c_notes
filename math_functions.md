# Math functions

Note: make sure to include the `<math.h>` library.

```c
// Square root
double a = sqrt(9); // -> 3.000000

// Power
double b = pow(2, 4); // -> 16.000000

// Round
int c = round(3.14); // -> 3

// Ceiling (round up)
int d = ceil(3.14); // -> 4

// Floor (round down)
int e = floor(3.99); // -> 3

// Absolute value
double f = fabs(-100); // -> 100.000000

// Logarithm
double g = log(3); // -> 1.098612

// sin, cos & tan
double h = sin(45); // -> 0.850904
double i = cos(45); // -> 0.525322
double j = tan(45); // -> 1.619775
```
